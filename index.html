Overview of the “Note Application” Project

1. Technologies Used:

  • • Java: Main programming language.
  • • Spring Boot: Main framework for developing the application.
  • • Spring Security: Manages user authentication and authorization.
  • • Spring Data JPA: Manages database connections and queries.
  • • Thymeleaf: Template engine for creating user interfaces.
  • • Bootstrap: CSS framework for designing user interfaces.
  • • MySQL: Relational database to store data.
  • • AWS S3: File storage.
  • • Jakarta Servlet: Manages sessions and HTTP requests.
  • • Maven: Project and dependency management tool.
  • • Excel: Export data to Excel files.
  • • Lombok: Reduces boilerplate code in model classes.
  • • Google Authenticator: Two-factor authentication (2FA).
  • • JavaMail: Sends notification emails.
  • • Avatar Capture: Captures and stores user avatars.
  • • Scheduled Tasks: Schedules the deletion of invalid or expired tokens.

2. Main Features of the Application:

Users (User):
  • - Register account: Users can register a new account.
  • - Login and logout: Users can log in and log out of the system.
  • - Edit personal information: Users can update their personal information.
  • - Manage personal notes: Users can create, edit, and delete personal notes.
  • - Two-factor authentication (2FA): Users can enable 2FA through Google Authenticator.
  • - Password recovery: Users can recover their password via email.
  • - Account activation: Activate account via email.
  • - Email notifications: Send email notifications for account updates or deletions.
  • - Avatar capture: Feature to capture and store avatars.
Group Management (Group):
  • - Create and manage groups: Create, delete, and update group information.
  • - Join and leave groups: Join groups with a group code or leave groups.
  • - Manage group members: View, block, unblock, and remove members.
  • - Manage group notes: Create, edit, delete, and search group notes.
  • - Export group notes to Excel files.
Trash:
  • - Manage trash: Temporarily store deleted notes, restore notes from trash, and permanently delete notes.
Admin Page:
  • - Manage users: View, edit, delete users, and send email notifications.
  • - Manage groups: View, edit, delete groups.
  • - Reports and statistics: View reports and statistics on users, groups, and notes.

4. User Pages and Interfaces:

Users:
  • - Home page: Displays general information and personal notes.
  • - Login/registration page: Allows users to log in or register a new account.
  • - Edit personal information page: Update personal information.
  • - Personal notes page: Create, edit, delete, and search personal notes.
  • - Account management page: Manage account information, upgrade to VIP, view notifications.
Groups:
  • - Group selection page: Displays a list of groups the user has created and joined.
  • - Join group page: Join groups with a group code.
  • - Create group page: Create a new group with an automatic group code.
  • - Group details page: Displays detailed information about the group, including member list and group notes.
  • - Group notes page: Displays a list of group notes and allows searching notes by content.
  • - Export group notes to Excel files.
Payments:
  • - Payment page: Select VIP package, enter discount code, and choose payment method (Momo, bank transfer).
  • - Payment confirmation page: Displays when the user completes the payment.
Email Notifications:
  • - Notification when updating personal information: Send email notification when personal information is updated.
  • - Notification when account is deleted: Send email notification when account is deleted.

5. Important Functions:

  • - Two-factor authentication (2FA): Using Google Authenticator.
  • - Password recovery via email.
  • - Account activation via email.
  • - Send email notifications to all or selected accounts.
  • - Notification of account deletion/update via email.
  • - Capture and store avatars.
  • - Schedule the deletion of invalid or expired tokens.
